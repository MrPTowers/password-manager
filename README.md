### Purpose
---
The purpose of this program is to provide a command line password manager with encryption.
A user is able to insert and retrieve passwords based on the service or app the password belongs to. A user is also able to generate random secure passwords for use.

### How to Run
---
Use: python3 app.py

The master password must have at least:
1. 10 characters
2. One uppercase letter
3. One lowercase letter
4. One digit
5. One special character

### Applied concepts from class
---

- Input validation: There are input validations each choice prompt as well as the master password input
- Error handling: Try-Excepts are used to handle errors such as json corruption
- Regex: Regular expressions are used for the master password input validation
